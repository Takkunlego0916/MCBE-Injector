# MCBE Injector

Minecraft Bedrock Edition用の軽量なDLLインジェクターです。

## 特徴
- **爆速:** C++ネイティブ実装による高速なインジェクション。
- **GUI:** シンプルで使いやすいインターフェース。
- **記憶機能:** 最後に使用したDLLパスを自動で保存します。
- **最新対応:** 最新のWindows版マイクラに対応。

## 使い方
1. [Releases](リンク) から最新の `MCBE Injector.exe` をダウンロードします。
2. マイクラを起動します。
3. インジェクターを起動し、DLLを選択して `Inject` を押します。

## 開発環境
- MinGW-w64 (g++)
- Windows API
